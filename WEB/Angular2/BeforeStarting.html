Index.html will be in the my-proj(or any other name) folder in src folder

BEFORE STARTING:
- install NodeJS
- install GIT from git-scm.com
- https://github.com/angular/quickstart
- copy clone line (git clone https://github.com/angular/quickstart my-proj)
- right click on project folder "GIT Bash here"
- copy the clone line, change "my-proj" to project name
- open that folder ^
- right click GIT Bash here
- npm install
- To run the project type => npm start
- refresh page for changes

- main focus on : project\src\app\app.component.spec.ts
- ^(compiled to JS in app.component.js) - main focus on main.js (same folder)
